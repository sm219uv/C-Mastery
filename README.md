A collection of **13 C programming challenges** designed to sharpen core skills ‚Äî from loops and conditionals to bitwise operations and number theory.

Each file contains a skeleton with a **goal** and **skill description**. Your job: implement the solution.

## Challenges

| #  | File                | Challenge                              | Key Skill                          |
|----|---------------------|----------------------------------------|------------------------------------|
| 01 | `01_stars.c`        | Build a star pyramid                   | Nested loops, output formatting    |
| 02 | `02_palindrome.c`   | Check if a number is a palindrome      | While loop, modulo                 |
| 03 | `03_armstrong.c`    | Armstrong number check                 | Math logic, digit separation       |
| 04 | `04_strong.c`      | Strong number check                    | Loops + factorial                  |
| 05 | `05_prime.c`       | Prime number check                     | Optimized loops, boolean flags     |
| 06 | `06_no_plus.c`     | Add two numbers without `+`            | Bitwise XOR, AND, Shift            |
| 07 | `07_half_adder.c`  | Half adder circuit simulation          | Low-level hardware, bitwise logic  |
| 08 | `08_fibonacci.c`   | Print Fibonacci sequence               | Iteration, variable swapping       |
| 09 | `09_floyd.c`       | Draw Floyd's triangle                  | Counters in loops, formatting      |
| 10 | `10_bin2dec.c`     | Binary to decimal conversion           | Number systems, base 2             |
| 11 | `11_power.c`       | Power calculation (no `pow()`)         | Custom math function               |
| 12 | `12_leap.c`        | Leap year check                        | Complex conditions (`if`, `&&`, `\|\|`, `%`) |
| 13 | `13_perfect.c`     | Perfect number check                   | Finding divisors, sum accumulation |

## How to Compile & Run

```bash
gcc -o challenge 01_stars.c
./challenge
```
